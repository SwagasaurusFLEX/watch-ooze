// what this is

Every validator on staccana is registered with the on-chain validator-subsidy program. The federation pushes uptime, delegated stake, and vote-count metrics every epoch. The program reads those metrics, computes a pro-rata weight, and pays validators out of the genesis treasury.

This page reads the program's state directly — ValidatorRegistry gives the list of registered identities, then one ValidatorRecord PDA per validator carries the metrics. No middleman.
